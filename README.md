# GIF转换工具

这是一个视频转gif的小工具，基于python实现，无论是Windows还是Mac都能用哦

不过不管是Windows还是Mac，都需要装个anaconda





## 工具使用方法

* 拷贝源码至本地：

    ```bash
    git clone ...
    cd gif_generate
    ```

* 安装项目依赖，并切换conda环境：

    ```bash
    conda env create -f env.yml
    conda activate gif_generate
    ```

* 运行该工具

    ```bash
    python manager.py runserver
    ```

* 打开浏览器，地址栏输入: localhost:8000，然后尽情使用
